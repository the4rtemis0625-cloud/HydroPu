# **App Name**: AquaSense

## Core Features:

- Real-time Monitoring Dashboard: Display current pH, water temperature, air temperature, humidity, and nutrient EC/TDS readings from the hydroponics system in a clear, real-time dashboard.
- Historical Data Trends: Visualize sensor data over customizable time periods (e.g., last 24 hours, 7 days, 30 days) using interactive charts and graphs to identify trends.
- Parameter Thresholds & Visual Alerts: Allow users to set custom ideal ranges for each environmental parameter and provide immediate visual indicators on the dashboard when a reading falls outside these defined thresholds.
- AI Growth Optimization Tool: Based on live and historical sensor data and user-defined crop type, an AI tool provides tailored recommendations for parameter adjustments to maintain optimal conditions for plant growth.
- User Authentication: Secure user login and account management to protect monitoring data and personalize settings.

## Style Guidelines:

- Primary color: A deep, invigorating green (#1F7348) to represent healthy plant growth and vitality, suitable for a light background.
- Background color: A very light, desaturated green (#E9F7EC) providing a clean, fresh canvas that complements the primary color and reflects natural purity.
- Accent color: A vibrant aqua (#4ED2C6) that suggests clean water and technological precision, used for interactive elements and highlights.
- Body and headline font: 'Inter' (sans-serif), chosen for its excellent readability, modern appearance, and neutrality, making data presentation clear and objective.
- Utilize a suite of clean, minimal line icons to represent sensor parameters (e.g., pH scale, water droplet for humidity, leaf for growth), enhancing visual understanding without clutter.
- A responsive, modular dashboard layout with clearly defined cards for each sensor reading, historical charts, and AI recommendations, ensuring intuitive navigation and optimal data presentation on various screen sizes.
- Subtle, fluid transitions for data updates on the dashboard and chart interactions, providing a smooth and engaging user experience without distracting from critical information.
